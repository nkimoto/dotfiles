## .vimrc

### Install NeoBundle
1. Make `.vim/bundle` directory  

        $ mkdir -p ~/.vim/bundle


1. clone `neobundle.vim` to `.vim/bundle`

        $ git clone git://github.com/Shougo/neobundle.vim ~/.vim/bundle/neobundle.vim

1. Start up vim

        $ vim
